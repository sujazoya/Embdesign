Adds an advanced search bar with filters for products.

== Description ==
This plugin provides an advanced search bar with dropdown filters for:
- Product title
- Design code (custom field)
- Categories
- Tags
- Custom taxonomies (area, stitches, etc.)

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through 'Plugins' in WordPress
3. Use the shortcode `[advanced_product_search]` in any page or widget
