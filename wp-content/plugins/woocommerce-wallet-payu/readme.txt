Adds wallet functionality with PayU payment gateway to WooCommerce.

== Description ==

This plugin adds a wallet system to WooCommerce with PayU payment gateway integration. Customers can:

* Add funds to their wallet using PayU
* Pay for orders using wallet balance
* View transaction history
* Use partial wallet payments

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woocommerce-wallet-payu` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure the PayU settings under WooCommerce > Settings > Payments
4. The wallet will appear in the My Account section

== Frequently Asked Questions ==

= How do customers add funds to their wallet? =

Customers can add funds by visiting the Wallet section in their My Account page.

= Can customers use wallet balance for partial payments? =

Yes, if the wallet balance is less than the cart total, customers can choose how much to pay from their wallet.
