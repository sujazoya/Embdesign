A WooCommerce extension for custom design orders with document uploads and designer proposals.

== Description ==

The Design Order plugin allows customers to submit custom design requests through a form with multiple file uploads. 
Sellers/designers can then submit proposals for the design work, and once approved, can upload the final design files.

Key Features:
- Custom design order submission form
- Multiple file uploads
- Designer proposals with pricing
- Order status tracking
- Secure file delivery after payment

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/the-design-order` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. The plugin will create necessary pages automatically

== Frequently Asked Questions ==

= What file types are allowed for upload? =

The plugin allows JPG, PNG, GIF, PDF, DOC, XLS, PPT, PSD, AI, EPS, ZIP, and RAR files.

= How do customers access their completed designs? =

After a designer uploads the final files and the customer completes payment, the files will be available for download from their account page.
