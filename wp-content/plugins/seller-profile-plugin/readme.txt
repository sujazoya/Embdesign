nstallation Instructions
Create a folder named seller-profile-plugin in your WordPress 
plugins directory (wp-content/plugins/)

Create all the files with the exact names and content as shown above

Zip the folder and upload it via WordPress admin, or
 simply activate it directly if you created it on the server

The plugin will automatically create a "Seller Profile" 
page with the [seller_profile] shortcode

The page will be accessible at /seller-profile/

Features
Complete seller profile system

Profile picture upload

Username change functionality

Product management (edit/delete)

Earnings calculation

Responsive design

Translation ready

Works with WooCommerce

No dependencies (except WooCommerce)

The plugin is completely standalone and doesn't require any modifications 
to your theme or other plugins. Just install and activate it.
